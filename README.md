# Руки Вверх! — Учебный сайт (тематика)

Файлы в проекте:
- index.html — главная страница.
- styles.css — стили оформления.
- script.js — скрипт для загрузки и вывода CSV-данных.
- data/tracks.csv — пример CSV с треками.
- README.md — инструкция.

## Запуск (локально)
Рекомендуется запускать через локальный HTTP-сервер, например:
```
python -m http.server 8000
```
и открыть http://localhost:8000

Если открывать index.html напрямую, браузер может запретить загрузку локального CSV (fetch). В этом случае увидите встроенный пример в скрипте.

## Примечание по авторским правам
На странице используются названия известных песен и ссылки на поиски/платформы.
